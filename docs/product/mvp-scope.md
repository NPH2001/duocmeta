# MVP Scope

## In Scope

* Home page
* Product listing
* Product detail
* Category pages
* Search
* Cart
* Checkout
* Order success page
* Customer account
* Order history
* Admin product management
* Admin order management
* Coupons basic
* CMS pages/blog basic
* SEO metadata

## Out of Scope

* Marketplace multi-vendor
* Loyalty program advanced
* ERP integrations
* Native mobile apps
* AI recommendations advanced
* Multi warehouse advanced routing

## User Roles

* Guest
* Customer
* Admin
* Catalog Manager
* Order Manager
* Content Editor

## Core Flows

1. Browse products
2. Add to cart
3. Checkout
4. Pay / COD
5. View order history
6. Admin manages catalog
7. Admin manages orders

## Success Metrics

* User can complete purchase end-to-end
* Admin can manage catalog and orders
* Public pages indexable by search engines
